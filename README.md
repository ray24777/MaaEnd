## ✨ 功能一览

- 🎁 **一键预设场景**：日常全套、快速领奖、挂机辅助、跑商倒卖、灵素养成……多种常用场景预设一键切换，省去繁琐配置！✨
- 🎮 **实时开荒辅助**：开荒小帮手上线！实时识别画面，按需补上关键操作，让你专心打就完事了 ✨
    - **自动剧情**：一键跳过对话、智能选分支、快速过剧情，爽就完事了！（当然你想慢慢看也行~）
    - **交互处理**：自动点指引光标（老普之手）、翻说明书、点点点继续、关档案语音，省心不手忙脚乱！
    - **辅助战斗**：自动普攻、放技能、开大招、打连携，还能锁定目标集火，战斗从此变得轻松写意✨
    - **自动拾取**：看到闪光就捡，绝不放过任何掉落物和采集点，资源收集小能手就是你！
    - **便捷操作**：秒速传送确认、自动滑滑索，赶路效率 MAX！
    - **顺手解拼图**：开荒路上遇到拼图？交给 AI 秒解（也可以直接开「🧩自动解拼图」循环）😎
    - **自动拍一拍**：驮兽也安排上，和自动拾取绝配（建议拍完迅速换🐮）🐾
- 🧩 **拼图大师**：AI 火眼金睛，谜题秒解！单次/循环/演示三模式（学习一下大佬怎么解题 😎）。
- 💎 **基质识别**：智能识别基质词条，自动筛选保留最优词条，杂质一键清理，再也不用一个个看属性了！✨
- 🔫 **武库升级 & 组装**：武器打磨升级、齿轮部件组装自动化，解放双手，轻松提升战力！💪
- 💰 **半自动倒卖**：跑商专家上线！自动识别最高利润货物并购买、自动进好友飞船；支持自动切区（也可以关掉），售卖这一步目前仍需你亲自出手~ 💸
- 🛒 **一键售卖产品**：各据点产品自动兑换调度券，点点点就交给它吧！
- 💳 **信用点购物**：自动购买信用商店物品，设置好想要的商品列表，挂机购物两不误！
- 🐌 **库存转移**：帝江号仓库跨区搬家，电池/矿/砂叶一键挪窝，囤囤鼠狂喜！
- 🔨 **批量简易制作**：简易制作选好道具，一键批量开搓（记得先把背包腾一腾）🔨
- 📐 **蓝图搬运工**：建筑党福音！批量导入蓝图，智能解析分享码/杂乱文本，工业流水线说建就建，效率拉满！
- 🎁 **基建任务**：自动领取基建产物、补货，线索收取/放置一条龙；线索交流也能安排（但不太推荐乱开）🎁
- 👥 **批量加好友**：一键批量发送好友请求，再也不用一个个手动输 UID 啦！🤝
- 📅 **日常自动化**：日常奖励（任务/通行证/邮件/暂存区）、模拟空间奖励、百科红点、简制手册领取、Baker 嘴替、好友拜访……一键清空！✅
- 🚚 **委托小秘书**：送货任务自动接取转交（可按地区开关），委托列表还能「抢单」~ 🚚
- 💻 **多端适配**：支持 PC 端 (Win32 API) 与安卓端 (ADB)，无论你在哪个平台，我们都能陪你摸鱼~ 🐟
